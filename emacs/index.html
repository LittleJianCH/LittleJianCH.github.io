2.1.2 Package Manager

I use use-package as package manager. Maybe I will change to manage them by gitsubmode. 

(require 'package)
(setq package-archives '(("gnu"   . "http://elpa.emacs-china.org/gnu/")
                         ("melpa" . "http://elpa.emacs-china.org/melpa/")))
(package-initialize)  

(unless (package-installed-p 'use-package)
  (package-refresh-contents)
  (package-install 'use-package))

(setq use-package-always-ensure t)

(require 'use-package)

2.2 Editor

2.2.1 Evil

I'm a Vimer before, so evil is very important. 

(use-package evil-leader
  :config
  (global-evil-leader-mode))

(use-package evil
  :after evil-leader
  :config
  (evil-mode t)
  (dolist (p '((eaf-mode . emacs)
               (eshell-mode . emacs)))
    (evil-set-initial-state (car p) (cdr p))))

2.3.5 Fullscreen

I like to start emacs in fullscreen. 

(defun fullscreen ()
  (interactive)
  (set-frame-parameter nil 'fullscreen
                       (if (frame-parameter nil 'fullscreen) nil 'fullboth)))
(fullscreen)

2.3.7 Startup Screen

(use-package dashboard
  :ensure t
  :config
  (dashboard-setup-startup-hook)
  (setq dashboard-set-file-icons t
        dashboard-set-heading-icons t
        dashboard-startup-banner 1
        dashboard-footer-messages '("Enjoy coding in emacs!"))
  (defun dashboard-goto-recent-file ()
    "Go to recent files."
    (interactive)
    (funcall (local-key-binding "r")))
  (evil-define-key 'normal dashboard-mode-map
    "r" 'dashboard-goto-recent-file))

2.4.2 Ivy

When you're in evil-normal-state, you can use ; to type emacs-command and use : (as S-;) to type vim-command. 

(use-package ivy
  :hook (after-init . ivy-mode)
  :bind
  (:map evil-normal-state-map
        (";" . execute-extended-command))
  :config
  (setq ivy-use-virtual-buffers t))
